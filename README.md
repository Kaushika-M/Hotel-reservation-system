Hotel Reservation System<br>
Project Description<br>
The Hotel Reservation System is a web-based application that allows customers to reserve hotel rooms online. Users can create an account, log in, search for available rooms based on their preferred dates, view room details, and complete their bookings through a payment page.

The system also provides an admin panel where hotel staff can manage rooms, monitor reservations, update room availability, and view customer booking information.

The main objective of this project is to simplify the hotel booking process, reduce manual work, and provide customers with a fast and convenient reservation experience.

Objectives
Allow customers to register and log in.
Display available rooms based on selected dates.
Enable online room booking.
Provide a payment confirmation page.
Allow customers to view booking history.
Enable administrators to manage rooms and reservations.

Technologies
Backend: FastAPI
Frontend: HTML, CSS, JavaScript
Database: SQLite
Template Engine: Jinja2
